# 제어기 선택

제어기의 종류에 따라 RoboPlus Task에서 사용 가능한 명령문 및 파라미터가 다릅니다. 따라서, 프로그램 작성 전에 제어기를 선택해야 합니다.

![img](/assets/images/sw/rplus1/task/controler_select.png)

`Note` RoboPlus Task에서 특정 제어기를 대상으로 프로그램을 작성 중에 다른 제어기로 변경하면, 호환되지 않는 명령 및 파라미터로 인해 룰 체크시 에러가 발생할 수 있습니다. {: .notice}

`Note` 기존에 작성된 프로그램 코드에서 제어기의 종류를 바꾸는 경우 호환되지 않는 명령문과 파라미터를 적절히 수정해 주어야 합니다.{: .notice}
